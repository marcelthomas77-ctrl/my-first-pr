# my-first-pr

A practice repository for learning how to open a pull request on GitHub.

## What is this?

This repo exists so I can practice the full pull request workflow:
branching, committing, pushing, and opening a PR.

## Workflow

1. Create a branch for your change.
2. Commit the change with a clear message.
3. Push the branch to GitHub.
4. Open a pull request and describe what changed and why.

## Status

This is a personal learning project, not intended for production use.
